⌨ Keyboard Shortcuts

Esc Toggle Sindhi / QWERTY mode
Shift Type shifted characters
Caps Toggle caps lock
Ctrl+C Copy text

This Sindhi Keyboard enables you to easily type Arabic Sindhi online without installing a Sindhi keyboard. You can use your computer keyboard or mouse to type Sindhi letters with this online keyboard.

Pressing Esc on the Sindhi keyboard layout will toggle the mouse input between virtual QWERTY keyboard and virtual Sindhi keyboard. Pressing Esc on your keyboard has the same function.

Press Shift for additional Sindhi letters that are not visible on the keyboard.
